# Mapping reference data

Catalogues for **DIN 276 Kostengruppe → BIP.BSABe** work and **ContractID** selector design.

## Regenerate

```bash
cd ifcpatch-worker

# BIP typbeteckningar.json + DIN 276 summary (public URLs)
python3 scripts/export_code_library_reference.py

# DIN 276:2018 codes from WEKA/SIRADOS comparison PDF (all ~320 KG codes)
python3 scripts/parse_din276_weka_pdf.py

# KG 300 slice + OpenConstructionERP partials (Revit/BIM, UI, CWICR prefixes)
git clone --depth 1 https://github.com/datadrivenconstruction/OpenConstructionERP /tmp/OpenConstructionERP
python3 scripts/extract_din276_ocerp_kg300.py --ocerp /tmp/OpenConstructionERP

# Naviate BSAB 96 HUS keynotes (local txt — see below)
python3 scripts/prep_bsab_naviate_reference.py
python3 scripts/generate_nobel_bsabwr_from_bsabe.py   # after nobel_a1_kostengruppe_bsabe.py exists
```

## Naviate source files (you added)

| File | BSAB table | BIP property |
|------|------------|--------------|
| `BSAB 96 Byggdelar HUS.txt` | Byggdelstabell | **`BIP.BSABe`** |
| `BSAB 96 Produktionsresultat HUS.txt` | Produktionsresultat (AMA) | **`BIP.BSABwr`** (not BSABe) |
| `BSAB 96 HUS.txt` | Combined export | Same split at row `A SAMMANSATTA PRODUKTIONSRESULTAT` |

Origin: Naviate Library SE → BSAB Keynotes (Revit). Encoding: **Windows cp1252** (script auto-detects).

| File | Use |

| **`bsab96_bsabe_quick_reference.txt`** | Human-readable BIP-style codes only (27.x, 43.CB, 63, …) |
| **`din276_prefix_bsab_hints.tsv`** | Curated DIN 3-digit prefix → suggested BSABe + Naviate title |
| **`din276_prefix_bsabwr_hints.tsv`** | Curated DIN 3-digit prefix → suggested BSABwr (AMA) |
| **`nobel_bsabe_validation.txt`** | Checks `nobel_a1_kostengruppe_bsabe.py` codes vs Naviate byggdelar |
| **`nobel_bsabwr_validation.txt`** | Checks `nobel_a1_kostengruppe_bsabwr.py` codes vs produktionsresultat |

### Nobel validation note

After prep, open `nobel_bsabe_validation.txt` (expect **20/20** exact matches).

Former **`32.G`** (invalid in Naviate HUS) was replaced in `nobel_a1_kostengruppe_bsabe.py`:

- DIN **361** structural → **27.G**
- DIN **363** roof covering → **41.C**
- DIN **364** roof lining → **41.D**

## Workflow: new Kostengruppe string

1. Parse DIN prefix from IFC string (`342 …` → `342`).
2. Check **`din276_prefix_bsab_hints.tsv`** for a starting BSABe.
3. Confirm title in **`bsab96_bsabe_quick_reference.txt`** or `bsab96_byggdelar.tsv`.
4. Cross-check **`bip_typbeteckningar_bsabe_only.txt`** for TypeID hints.
5. Add exact string to gitignored `nobel_a1_kostengruppe_bsabe.py`.
6. Re-run `prep_bsab_naviate_reference.py` → refresh `nobel_bsabe_validation.txt`.
